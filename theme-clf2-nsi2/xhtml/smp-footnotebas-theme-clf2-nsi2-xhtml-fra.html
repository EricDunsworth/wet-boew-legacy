Exemple simple - Notes de bas de page accessibles - Thème NSI 2.0

Vue d'ensemble

Le sous-projet Notes de bas de page accessibles vise à mettre en place une méthode cohérente et facile d’accès pour répertorier les notes de bas de page dans l’ensemble des sites Web du gouvernement canadien. L’idée force qui sous-tend cette solution consiste à classer les notes de bas de page dans une liste de définitions à l’intérieur d’une section spécifique. Un exemple de ceci peut être trouvé dans la section Notes de bas de page. Des styles CSS servent à disposer les notes de bas de page et à dissimuler les aides à la navigationNote de bas de page 1.

Avantages

  • Conformes à WCAG 2.0 AANote de bas de page 2
  • Approche d'amélioration progressive
  • Soutien pour Firefox, Opera, Safari, Chrome et IE 7+
  • Soutien pour l'anglais et le français
  • Mise en page et conception configurable

Utilisation recommandée

  • Exécuter les notes de bas de page dans les pages WebNote de bas de page 3
