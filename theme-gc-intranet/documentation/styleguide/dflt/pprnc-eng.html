Default appearance of elements

The default elements are self-styling and do not require the use of CSS class names to achieve their appearance. The default.css file generates the necessary indenting, sizing, colour and so on. However, if there is need to override the default appearance, refer to the CSS: Snapshot of class names for a comprehensive list of available classes.
