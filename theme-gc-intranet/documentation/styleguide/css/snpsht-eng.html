CSS: Snapshot of class names

You can combine and layer many of these classes on the same element, for example, <div class="span-2 margin-top-none background-light border-span-2 color-accent float-right">. This offers a nearly unlimited number of unique presentation options, while still adhering to a strict CSS strategy.

These CSS names use American English to match standard CSS coding (color instead of colour).
