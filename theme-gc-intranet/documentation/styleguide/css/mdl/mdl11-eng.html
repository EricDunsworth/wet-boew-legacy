Overview of module-summary

Introducing CSS class:

module-summary

The module-summary class features content summaries, recaps and reviews. This module works with span-2, span-3, span-4, span-5 and span-6.

In general, modules are specific CSS classes that work within the grid system and create an entire presentation style. While all modules are found here, this particular module creates the following effect:

Summary: Example text

Before using this module, we recommend that you have an understanding of grid CSS.

Points of compliance

Use this module only for its intended purpose. This ensures predictable outcomes to any changes made to the CSS, and that the user experience is consistent.

Correct use of module-summary

When using this module, always follow these basics:

  • Use to feature summaries, recaps and reviews.
  • Use always within a grid system (span-2 to span-6). For example:<div class="span-2 module-summary">. The flexibility of the grid system allows you to place these modules anywhere within the system (they do not need to left-align).
  • Use this module sparingly and strategically. Overuse of the design creates visual clutter on the page.
  • Start each module-summary with trigger words such as Summary, Review or Recap. Make sure to wrap these trigger words in the <strong> element.

Incorrect use of module-summary

Do not do the following:

Use this module in a manner that contradicts its intend use.
