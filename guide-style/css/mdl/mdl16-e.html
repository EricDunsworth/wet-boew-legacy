Overview of module-form-fluid

Introducing CSS classes:

  • module-form-fluid
  • module-form-fluid-none

The module-form-fluid class stretches form elements to the full width of the grid cell. This module works with span-2, span-3, span-4, span-5 and span-6.

In general, modules are specific CSS classes that work within the grid system to create an entire presentation style. While all modules are found here, this particular module creates the following effect:

Personal Information

Before using this module, we recommend that you have an understanding of grids.

Points of compliance

Use this module only for its intended purpose. This ensures that any changes made to the CSS have predictable outcomes, and guarantees a consistent user experience.

Correct use of module-form-fluid

When using this module, always follow these basics:

  • Use module-form-fluid for all your form layout needs, as the most usable forms always have the label above the input box.
  • Use only within a grid system (span-2 to span-6). For example: <div class="span-2 module-form-fluid">. The flexibility of the grid system allows you to place these modules anywhere within the system.
  • For elements that you do not want to stretch to the full width of the grid cell, use the class module-form-fluid-none.

Incorrect use of module-form-fluid

Do not do the following:

Use this module in a manner that contradicts its intend use.
