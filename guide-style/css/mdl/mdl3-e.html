Overview of module-billboard

Introducing CSS class:

module-billboard

The module-billboard class features a list of links to specialty and newsworthy pages. It forms part of the right menu. This module works with span-2.

In general, modules are specific CSS classes that work within the grid system to create an entire presentation style. While all module CSS are found here, this particular module creates the following effect:

Before using this module, we recommend that you have an understanding of grids.

Source code and appearance options

Add a background image to your billboard by following these steps:

  1. Add this JavaScript to the editable area in the <head> tag of the page the billboard appears on: 
    
<script src="http://infozone/wet20/js/plugins/font-replacement.js" type="text/javascript"></script>
  2. Locate the desired background image from the options below. Its class name (for example,. billboard-32) appears in front of that image.
  3. Apply that class to your billboard item. For example: 
    <li><a href="#" class="billboard-32">Billboard link</a></li>

The module-billboard has several layout options (all appearing on the right). Next to each example is the corresponding source code that you can select and copy.

Floating technique

There are times when you want to place the module on the right side of a page, and have the neighbouring content wrap around and below it. To achieve this, apply a float-right to the module, but not in a grid cell. Then place the content after the module.The content then freely wraps around the module and fills all the available white space.

Points of compliance

Use this module only for its intended purpose. This ensures predictable outcomes to any changes made to the CSS, and that the user experience is consistent.

Correct use of module-billboard

When using this module, always follow these basics:

  • Only feature links to short-term specialty pages (announcements, general interest) and short-term newsworthy pages (stories, events and happenings).
  • Place on the right side of the Web page. You can combine with it with a float-right, as needed.
  • Use within a grid system, but only within a span-2. For example: <div class="span-2 module-billboard">.
  • Include the class span-footer <div class="span-2 module-billboard span-footer">, unless you stack this module with other modules. In that case, only the final module has the span-footer class.
  • Place on any Web page, as needed. This module can appear multiple times throughout the page.
  • Include a link at all times.
  • Start each list with the heading "Billboard"( <h2> — <h6> all work).
  • Place stacked modules (children) inside a parent span-2 . This parent contains these grid modules so they don't float to other spots on the page. Anytime you establish a grid parent and grid child relationship, you normally have to apply the row-end class to each child module. However, the CSS in this template forecasts this specific relationship and accounts for it. So in this case, there is no need to apply the row-end class. If you apply the row-end class out of habit, there is no harm.
  • Remove the billboard item after 90 days. If you do not constantly rotate and update billboard items, users develop what is known as banner blindness. That is, they tend to fully ignore the billboard content, including important messages.

Incorrect use of module-billboard

Do not do the following:

  • Use this module in a manner that contradicts its intend use.
  • Use more than one module-billboard in a stack.
  • Place permanent, static menu items in this module. This is only for temporary listings. Do not use this module as a graphical menu system to supplement or substitute your primary navigation.

Note: Background images can accommodate up to three lines of text that occupy the full width of the grid cell.
