GC style guide

Who should use this

Any Government of Canada site can choose to use this style guide.

The guide ensures that we use the design and scripting features as they were intended. This allows us to move forward in a structured and strategic way. Sites that do not adopt the style guide may find that their use of the templates, CSS and scripts is out-of-step with future versions of the code.

What are the perks?

Aside from a ton of guide-style, you get:

  • The ability to contribute to the style guide, as it is a collaborative product.
  • Structured reference material for your webmasters and authors to refer to.
  • An understanding of how the scripts and styles all fit together.
  • Get points of compliance for each CSS style and plug-in.

Psst...this is an all-or-nothing solution

There is no point in creating a local variant of this product and then changing this-and-that for your local site. If legitimate changes are needed, then give feedback to THIS product and we can look at changes on a global scale. Otherwise, the value of the style guide is quickly eroded when there are 100+ versions of it.
