Form interface plug-ins

WET plug-ins, widgets, and styles should be used according to the purpose and audience of the Web page and should contribute to, or enhance, the usability of the page and the users' experience. They should be used for their intended purposes, and never to compensate for bad page design or poor document or content structure.

Accessible CAPTCHA

Overview

This feature provides a jQuery plugin for adding an accessible challenge-response test to ensure that a response is not generated by a computer.

Examples:

  • DISTCHA proof of concept

Points of compliance

This plug-in is still in proof-of-concept stage. The existing CAPTCHA solution is inaccessible to people with disabilities and a "sliding alternative to CAPTCHA" called DISTCHA is being explored.

For more information, refer to the Accessible CAPTCHA project.

Auto-complete for text input fields

Overview

This feature provides auto-complete functionality to specific text input fields by dynamically displaying a list of words that match the user's input.

Examples:

  • Auto-complete for text input fields
  • Remplissage automatique des champs de saisie

Points of compliance

Use this feature only for its intended purpose. The recommended usage includes search input fields.

For more information, refer to Auto-complete for text input fields - Feature guide.

Checkbox list enhancement

Overview

This feature enhances the appearance of a checkbox list and makes it easier to use by visually grouping checkboxes together and making it easier to identify which checkboxes are selected and which are not.

Examples:

  • Checkbox list enhancement
  • Amélioration de liste de case à cocher

Points of compliance

Use this plug-in only for its intended purpose. This ensures a consistent user experience.

For more information, refer to Checkbox list enhancement - Feature guide.

De-selectable radio buttons

Overview

This feature enables radio buttons to be deselected.

Examples:

  • De-selectable radio buttons
  • Boutons radio désélectionnable

Points of compliance

Use this plug-in only for its intended purpose. It is recommended that this feature be use only in a section of a form where the user must select one option or none at all, using radio buttons.

For more information, refer to Deselectable radio buttons - Feature guide.

Form validation

Overview

This feature provides generic validation and error message handling for Web forms.

Examples:

  • Form validation
  • Validation des formulaires

Points of compliance

Use this plug-in only for its intended purpose. Note that enabling only client-side validation can leave a server open to malicious attacks.

For more information, refer to Form validation - Feature guide.

Slider control

Overview

This feature provides an accessible slider interface.

Examples:

  • Slider control
  • Commande de barre coulissante

Points of compliance

Use this plug-in only for its intended purpose.

For more information, refer to Slider control - Feature guide.
